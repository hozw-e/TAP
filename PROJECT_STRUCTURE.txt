│

│   ├── guardians/

WORKFLOW:
=========
1. Admin logs in via /admin/index.php
2. Login calls /api/auth/login.php
3. On success, redirect to /admin/dashboard.php
4. Dashboard checks session via /api/auth/check-session.php
5. Student scans NFC -> ESP32/Arduino sends to /api/nfc/scan.php
6. scan.php:
   - Checks if admin is logged in (session active)
   - If yes: logs attendance, sends SMS, returns success
   - If no: returns error "No admin session active"
7. Dashboard polls /api/attendance/recent.php every 3 seconds
8. Shows popup notification when new attendance detected
